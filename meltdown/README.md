# Meltdown/Spectre

https://cxsecurity.com/issue/WLB-2018010039

# HOW TO USE

#### On MacOS X
```
brew install cmake
mkdir build
cd build
cmake ..
make 
./meltdown
```

#### On Ubuntu
```
sudo apt-get install gcc cmake
mkdir build
cd build
cmake ..
make 
./meltdown
```

#### On Cent-OS / Amazon Linux
```
sudo yum install gcc cmake
mkdir build
cd build
cmake ..
make 
./meltdown
```

#### On Alpine
```
apk update
apk add gcc
apk add cmake
mkdir build
cd build
cmake ..
make 
./meltdown
```
